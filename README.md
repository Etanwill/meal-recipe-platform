# Community Meal-Ordering & Recipe-Learning Platform

A full-stack platform for ordering meals and sharing/learning recipes with community features.

## Features

### User Authentication
- Email/Password registration and login
- Secure password hashing
- User profiles

### Meal Ordering
- Browse meals by category, vendor, popularity
- Search and filter meals
- Shopping cart and checkout
- Order tracking
- Vendor dashboard

### Recipe Learning/Sharing
- Upload recipes with images/videos
- Step-by-step instructions
- Ratings and comments
- Save favorite recipes

### Community Features
- Follow cooks/vendors
- Share recipes/reviews
- Upvote/like system
- Leaderboard

## Tech Stack

### Backend
- Python Flask
- SQLAlchemy (SQLite/PostgreSQL)
- JWT Authentication

### Frontend
- React.js
- Material-UI
- Context API for state management

## Setup Instructions

1. Clone the repository
2. Setup Backend (see backend/README.md)
3. Setup Frontend (see frontend/README.md)
4. Run both servers
5. Access at http://localhost:3000

## API Documentation
Available at http://localhost:5000/api when backend is running
